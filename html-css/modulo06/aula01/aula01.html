Opções mais comuns de display

Por padrão o paragrado tem display com o valor block (ocupa uma linha inteira). inline (ocupa apenas o espaço do conteúdo) e inline-block (ocupa apenas o espaço do conteúdo, mas permite a definição de largura e altura) são outras opções de display. O display none é utilizado para ocultar um elemento
